Window Remote Version 1.0.0

Window Remote utilizes DisplayFusion Keybinds, Unified Custom Remotes and Windows 10 KeyBinds to give you full window control!

Controls and Keybinds


DisplayFusion

- Next Window "Ctrl + Win + X"
- Maximize Window "Ctrl + Win + M"
- Minimize Window "Ctrl + Win + N"

Unified Remote Additions - (STOCK)

- MouseCommand - Swipe 2 Fingers Down
- KeyboardControl - Bottom Right hand of Remote

Windows 10 Controls - (STOCK)

- Show Desktop "Win + D"
- Window Select "Win + TAB"
