WindowRemote.

WindowRemote is a custom remote designed to be used by Unified Remote that incorporates DisplayFusion and Windows 10 Keybinds to allow for window control all from your smartphone!

About this project

The point of this project was to be able to easily control and manipulate windows on my computer from my bed. Unified remote allows me to do that but I have to focus on dragging and dropping as well as finding the mouse. I figured that I could incorporate the basic Windows 10 Commands and DisplayFusion Window Commands and just build them into a custom unified remote control.

This isn't a huge project but it's something I would like to expand going forward and use to improve my coding and git skills. Thanks for taking a looking!
